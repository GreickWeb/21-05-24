1. O que é um banco de dados?

Um banco de dados é uma coleção organizada de informações ou dados, normalmente armazenados eletronicamente em um sistema de computador. Ele é controlado por um Sistema de Gerenciamento de Banco de Dados (SGBD), que permite a coleta, armazenamento, recuperação e manipulação eficiente dos dados. Os dados nos bancos de dados são modelados em tabelas com linhas e colunas, facilitando o processamento e a consulta.

2. Quais são os diferentes tipos de bancos de dados?

Banco de Dados Relacional (RDBMS)
Banco de Dados Não Relacional (NoSQL)
Banco de Dados em Memória
Banco de Dados Distribuído
Banco de Dados Orientado a Objetos

3. O que é um SGBD (Sistema de Gerenciamento de Banco de Dados)? Dê alguns exemplos.

Um SGBD é um software que permite a criação, gerenciamento e manipulação de bancos de dados.
Exemplos incluem MySQL, PostgreSQL, Oracle, MongoDB e Microsoft SQL Server.
